# Data Collection

## Overview

This folder contains sample datasets and planning materials used for the AI-Based Crosswalk Monitoring System.

The data will be used for:
- Vehicle detection
- Pedestrian detection
- Crosswalk analysis
- Risk evaluation

---

## Data Sources (Planned)

- Traffic video samples
- Public road surveillance datasets
- Simulated crosswalk scenarios

---

## Data Types

- Images (vehicles, pedestrians)
- Video frames
- Annotated detection labels (future work)
- Event logs (future work)

---

## Notes

At this stage of the project, data collection is in the planning phase.
Actual dataset integration will be implemented during the detection system development (Sprint 1).
